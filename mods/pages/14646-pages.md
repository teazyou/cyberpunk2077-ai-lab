## About this mod

The scanning no longer restricts you to walking only mode. You can now run, sprint or walk like normal while scanning and have a full control over your movement speed.

 Collections containing this mod 

 This mod is meant to be used in conjunction with other mods that disable the time dilation effect when entering the scanner mode. Such as[Scanner Time Dilation Optional 2.01](https://www.nexusmods.com/cyberpunk2077/mods/9671)(yes, it still works with the current 2.12a game version).

Check the requirements. Install using Vortex/Mod Organizer 2 or manually copy the files to the ...\r6\scripts folder.
