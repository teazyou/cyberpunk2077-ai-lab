**_About_🧐**

This mod does not change the ads.Fixes the problem of blurry ads by ensuring that ads are displayed with the best quality option.

Not only ads but also,😃

This mod makes this **quality improvement** ✨by addressing **at least 19 categories** of aspects of the game.

💠 Changes have been made to the .inkatlas file of all categories.

💠 **1.131**.inkatlas files in total.

💠 The "Highest quality resolution" option loads when you select a resolution above 2K, but I have observed that the game somehow sometimes loads lower versions as well.

✨With this mod, we prevent this problem with **quality improvement** by only loading the "Highest quality resolution"" option.

💠 I didn't include too many promotional images to avoid spoilers, each .inkatlas file is a different content in the relevant category, **so much more** has been done than what you see in the promotional images.
