 Collections containing this mod 

 Throwing weapons reload instantly without picking them up and/or won't force equip your next weapon if you spam throw.

You can find a redmod version of the mod here:[Better Throwing Knives and Weapons - Redmod](https://www.nexusmods.com/cyberpunk2077/mods/9543)

**Main files**

**BetterThrowingWeapons**Throwing weapons reload instantly without picking them up and/or won't force equip your next weapon if you spam throw. Be aware that when using this file on its own you will still get the red recovery circle when throwing the weapons. I left it enabled so that the mod is compatible with other mods like[Persistent Throwable Weapons](https://www.nexusmods.com/cyberpunk2077/mods/9432?tab=description).

Thrown weapons will disappear after the recovery cooldown or after picking them up if you don't use the Persistent Throwable Weapons mod, just like in the vanilla game.

**Optional files

BetterThrowingWeapons_DisableRecoveryCircle**This addon disables the red recovery circle and somewhat makes the knives persistent, meaning they won't disappear unless you go close to them to pick them up or until the game despawns them. Install this along with the main file. Incompatible with Persistent Throwable Weapons.

**DontForceEquipNextWeapon**

Use this if all you want to do is to stop the game from force equipping the next weapon when pressing the throw button while the weapon is on recovery cooldown. The main file already does this, this is for anyone that doesn't care about the other features of the mod.
